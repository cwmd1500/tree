Course Descriptions

This course gives the students a high-level overview of the skills and abilities necessary to become a front-end web designer and developer. Focus is on providing students with a bigger picture of what is involved with creating web and interactive media projects. Some terminology and technology will be introduced followed by real-world processes covering but not limited to web design, web host management, project workflow and version control.
